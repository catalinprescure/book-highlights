# Book Highlights

Highlights .txt archive from user's screenshots.

## Usage
- Save the image to screenshots/ archive directory
- Run the update script
- Run the program from console (index.js)
- Run the program from browser (public_html/)

~~~js
python update.py
~~~

~~~js
node index.js
~~~
